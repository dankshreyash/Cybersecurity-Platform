# APS – Cybersecurity Platform

A production-grade React application for a B2B SaaS cybersecurity platform with three screens: Login, Dashboard, and Active Scan Detail.

## Tech Stack

| Technology | Purpose |
|---|---|
| **React 19**  |
| **Tailwind CSS v4** | 
| **React Router v7** | Client-side routing |
| **Context API** | Global state (theme, toasts) |

## Setup Instructions

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/dankshreyash/Cybersecurity-Platform.git
cd assignment2

# Install dependencies
npm install
```

### Development

```bash
npm run dev
# Open http://localhost:5173
```

### Production Build

```bash
npm run build
npm run preview   # Preview the production build locally
```
